# Missed Prayers Calculator | حاسبة الصلوات الفائتة

A simple, Arabic-language web application that helps Muslims calculate the number of days of missed prayers, so they can plan to make them up (*qadha*).

---

## 📖 About the Project

This tool allows a user to enter the last date they prayed and calculates how many days of prayers they have missed — either up to today's date or up to a custom end date. Each missed day corresponds to five obligatory prayers (Fajr, Dhuhr, Asr, Maghrib, and Isha) that need to be made up.

The application is built entirely in a **single HTML file** with no external dependencies beyond a Google Fonts stylesheet, making it lightweight and easy to deploy anywhere.

---

## ✨ Features

- 📅 Select the last date you prayed to start the calculation
- 📆 Calculate missed days **up to today** with one click
- 🔢 Optionally calculate missed days **up to a specific date**
- ⚠️ Input validation with clear Arabic-language error messages
- 🌙 Clean dark-mode UI styled with the Nord color palette
- 📱 Fully responsive layout for desktop and mobile

---

## 🚀 Usage

1. Open `index.html` in any modern web browser (no server required).
2. Select the **last date you prayed** from the date picker.
3. Click **"الحساب حتى اليوم"** to calculate missed days up to today, **or**
4. Click **"الحساب حتى تاريخ محدد"** to reveal a second date picker, then choose an end date to calculate missed days up to that specific date.
5. The result is displayed instantly below the buttons.
